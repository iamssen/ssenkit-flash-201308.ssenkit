# 제공하는 API

- 비교
	- `equal()`
	- `equalMonth()`
	- `equalWeek()`
	
- 정보
	- `weekOfMonth`
	- `weekOfYear`
	- `dayOfYear`
	
- 복제, 이동
	- `clone()`
	- `getNextDay()`
	- `getPrevDay()`
	- `getFirstDayOfMonth()`
	- `getLastDayOfMonth()`
	- `getFirstDayOfWeek()`
	- `getLastDayOfWeek()`
	- `getNextMonth()`
	- `getPrevMonth()`
	- `getNextWeek()`
	- `getPrevWeek()`
	- `getNextYear()`
	- `getPrevYear()`
